## 📖 About This Repository

This repository contains an automated Bash script designed to instantly provision, configure, and complete the tasks required for the **"Develop and Secure APIs with Apigee X: Challenge Lab"**.

Instead of manually navigating the Google Cloud Console and Apigee interface, this script interacts directly with the Google Cloud SDK and Apigee APIs to handle the proxy configuration, routing, and security policies seamlessly.

---

## 🚀 Quick Start: Run in Cloud Shell

To execute the automated deployment, open your Google Cloud Shell terminal, copy the block below, and paste it to run the commands:

```bash
curl -LO https://raw.githubusercontent.com/saswatsubhransudev/Develop-and-Secure-APIs-with-Apigee-X-Challenge-Lab/refs/heads/main/saswatsubhransu.sh
sudo chmod +x saswatsubhransu.sh
./saswatsubhransu.sh
